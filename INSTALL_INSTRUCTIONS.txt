═══════════════════════════════════════════════════════════════════
  STEP 1: DOWNLOAD NODE.JS
═══════════════════════════════════════════════════════════════════

1. Open your web browser (Chrome, Edge, Firefox, etc.)
2. Go to this website: https://nodejs.org/
3. You'll see TWO buttons:
   - LTS (Recommended) ← CLICK THIS ONE (it's green)
   - Current
4. The download will start automatically
5. File name will be: node-v20.x.x-x64.msi

═══════════════════════════════════════════════════════════════════
  STEP 2: INSTALL NODE.JS
═══════════════════════════════════════════════════════════════════

1. Go to your Downloads folder
2. Find the file: node-v20.x.x-x64.msi
3. Double-click to run it
4. Installation wizard will open:
   
   Screen 1: Welcome → Click "Next"
   Screen 2: License Agreement → Check "I accept" → Click "Next"
   Screen 3: Destination Folder → Keep default → Click "Next"
   Screen 4: Custom Setup → Keep all checked → Click "Next"
   Screen 5: Tools for Native Modules → Keep default → Click "Next"
   Screen 6: Ready to Install → Click "Install"
   
   ⏳ Wait for installation (1-2 minutes)
   
   Screen 7: Completed → Click "Finish"

5. ✅ IMPORTANT: Make sure "Add to PATH" was checked during install

═══════════════════════════════════════════════════════════════════
  STEP 3: RESTART VS CODE
═══════════════════════════════════════════════════════════════════

1. Close VS Code completely (File → Exit)
2. Wait 5 seconds
3. Open VS Code again

═══════════════════════════════════════════════════════════════════
  STEP 4: VERIFY NODE.JS IS INSTALLED
═══════════════════════════════════════════════════════════════════

1. In VS Code, open Terminal (View → Terminal)
2. Type this command and press Enter:

   node --version

3. You should see something like: v20.11.0
4. Type this command and press Enter:

   npm --version

5. You should see something like: 10.2.4

✅ If you see version numbers, Node.js is installed correctly!
❌ If you still see "not recognized", restart your computer and try again.

═══════════════════════════════════════════════════════════════════
  STEP 5: INSTALL PROJECT DEPENDENCIES
═══════════════════════════════════════════════════════════════════

1. In VS Code Terminal, type these commands ONE BY ONE:

   cd "c:\Users\yash singh\OneDrive\Desktop\healthyfy"

   (Press Enter, then type:)

   npm install

2. Wait for installation (2-5 minutes)
3. You'll see a progress bar and many packages being installed
4. When done, you'll see: "added XXX packages"

═══════════════════════════════════════════════════════════════════
  STEP 6: RUN YOUR APP! 🚀
═══════════════════════════════════════════════════════════════════

1. In the same Terminal, type:

   npm run dev

2. Wait 5-10 seconds
3. You'll see:

   VITE v5.x.x  ready in XXX ms
   
   ➜  Local:   http://localhost:3000/
   ➜  Network: use --host to expose

4. Your browser will automatically open to: http://localhost:3000
5. You'll see your Healthyfy app running! 🎉

═══════════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Problem: "node is not recognized" after installing
Solution: 
  1. Restart your computer
  2. Open VS Code again
  3. Try the commands again

Problem: "npm install" fails
Solution:
  1. Delete the node_modules folder (if it exists)
  2. Run: npm install --force

Problem: Port 3000 is already in use
Solution:
  1. Press Ctrl+C in terminal to stop
  2. Run: npm run dev -- --port 3001
  3. App will open on port 3001 instead
